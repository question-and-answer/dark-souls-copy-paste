# Souls Copy / Paste Overlay

A Chrome extension that shows a **Dark Souls–style fullscreen overlay**
when you press **Ctrl/Cmd + C** or **Ctrl/Cmd + V** — complete with
dramatic red text, sound, and combo chaining.

> For fun. Not affiliated with FromSoftware.

---

## Features

- Fullscreen dark transparent overlay
- Souls-style red text
- Sound playback (user-provided)
- Ctrl/Cmd + C → funny “copy” messages
- Ctrl/Cmd + V → funny “paste” messages
- Combo system (COPIED ×3, ×4…)
- Click or press **Esc** to dismiss
- Customizable phrases, timing, volume

---

## Installation (Developer Mode)

1. Clone or download this repo
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder

## Configuration

Open the extension’s **Options** page to:
- change phrases
- adjust volume
- sync overlay duration to audio
- tweak combo timing

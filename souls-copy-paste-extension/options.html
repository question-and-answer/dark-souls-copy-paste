Souls Copy/Paste Overlay — Options

Triggers on Ctrl/Cmd + C and Ctrl/Cmd + V on any page. Rapid repeats bundle into combos like COPIED ×3.
Sound file should be inside the extension folder, default: sounds/souls_chime.mp3.

Example: sounds/souls_chime.mp3

Extra time after audio ends.

If unchecked, holding Ctrl+V will increment the combo.

Used when sync-to-audio is OFF (or as a minimum).

Optional. Some sites restrict paste events; keydown already works.

Font tip: if you have a font you’re allowed to use, add fonts/SoulsFont.woff2 and define @font-face in overlay.css.
